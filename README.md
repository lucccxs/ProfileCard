# Laboratório Front-End: Profile Card

Este repositório contém um trabalho prático desenvolvido durante o módulo de Front-End no SENAI, com foco em aplicar conceitos de lógica de programação na web.

O que pratiquei neste exercício:

Seleção de Elementos: Uso de querySelector e getElementById para acessar o HTML via JS.

Eventos: Implementação de addEventListener para capturar interações de clique e teclado.

Lógica de Temas: Criação de um sistema de "Modo Noturno" alternando classes CSS dinamicamente.

Validação: Verificação de inputs para evitar dados inválidos no cartão.
